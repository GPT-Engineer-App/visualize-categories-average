# visualize-categories-average

Create a web app for customers to visualize a dataset where they can categorize items and calculate their average values. The web app should allow users to group items into categories, calculate the average value for each category, and display the results.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/visualize-categories-average.git
cd visualize-categories-average
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
